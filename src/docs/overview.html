Menjangan-java

This is the reference for Menjangan Java.

Getting Started

Menjangan can't work standalone, you need have an implement of websocket. And Menjangan can work with all major framework : Spring, Micronaut, Quarkus

The main entry point to use Menjangan RPCP is simple as create RPCPHost Object. For example: 

    
        RPCPHost<WebSocketSession> rpcpHost = new RPCPHost<>("api.glexpress.id/ws", new MyRPCPHostHandler<>());

Versions and Compatibility

Version 1.0.0

Error Codes

Upon errors, the server will throw exceptions with error codes :

  • 400 — Invalid request parameters; Malformed requests
  • 401 — Authentication failure
  • 403 — Authorization failure
  • 404 — Object not found
  • 500 — Internal API error
  • 503 — Response temporarily unavailable; caller should retry later
